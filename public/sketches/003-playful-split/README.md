## Variant: Playful Split

### Design Stance
Split-screen hero with interactive product showcase, micro-animations, and personality — feels like a modern D2C brand (Casper, Article, Burrow). Prioritizes discovery and delight.

### Key Choices
- **Layout**: 50/50 hero split, rotatable product viewer, floating decorative shapes, tilted cards
- **Typography**: Space Grotesk (display) + Quicksand (body) — geometric, friendly, distinctive
- **Color**: Gradient accents (amber→orange), brand-color category chips, playful color swatches
- **Interaction**: Hero image switcher, color palette picker, scroll-reveal animations, horizontal carousels, style quiz teaser

### Trade-offs
- **Strong at**: Discovery, social sharing, style quiz conversion, mobile engagement
- **Weak at**: Accessibility (motion), performance (animations), information density

### Best For
- Younger homeowners, gift shoppers, social traffic
- Brands wanting "viral" shareability
- Quiz-driven personalization funnels
