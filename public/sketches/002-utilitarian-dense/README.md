## Variant: Utilitarian Dense

### Design Stance
Tool-first, high information density — like a well-organized workshop or Bloomberg terminal. Prioritizes speed, scanning, and decision-making over atmosphere.

### Key Choices
- **Layout**: Compact top bar, sticky category nav, 4-column product grid, minimal hero
- **Typography**: System font stack (SF Pro / Segoe UI / Roboto) — fast, familiar, no font load
- **Color**: Functional grays with amber as action accent only
- **Interaction**: Instant filters, load-more pagination, hover states for density

### Trade-offs
- **Strong at**: Comparison shopping, power users, mobile efficiency, conversion speed
- **Weak at**: Brand storytelling, emotional resonance, discovery

### Best For
- Returning customers who know what they want
- Mobile users on the go
- B2B / trade buyers comparing specs
