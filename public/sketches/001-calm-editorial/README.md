## Variant: Calm Editorial

### Design Stance
Content-first, airy layout with serif display typography — feels like a premium lifestyle magazine. Prioritizes storytelling and brand atmosphere over density.

### Key Choices
- **Layout**: Single-column hero, generous whitespace, asymmetrical grid for collections/products
- **Typography**: Cormorant Garamond (display) + Inter (body) — editorial, sophisticated
- **Color**: Amber/gold accent on warm dark hero, neutral grays elsewhere
- **Interaction**: Subtle hover lifts, fade-in overlays, smooth scroll animations

### Trade-offs
- **Strong at**: Brand storytelling, emotional connection, premium perception
- **Weak at**: Information density, quick scanning, mobile efficiency

### Best For
- First-time visitors exploring brand identity
- Content-forward audiences (design-conscious homeowners)
- Desktop/tablet browsing sessions
